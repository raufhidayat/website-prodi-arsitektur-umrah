# Prototype Website Program Studi Arsitektur FTTK UMRAH

Versi awal website statis Program Studi Arsitektur FTTK UMRAH untuk bahan presentasi dan penawaran kepada pihak program studi.

## Tujuan versi pertama

Homepage mengikuti pola umum website Program Studi Teknik Informatika FTTK UMRAH: navigasi bertingkat, hero section, PMB, profil singkat, status akreditasi, sambutan koordinator, statistik, berita, pengumuman, karya mahasiswa, kerja sama, dan footer.

Seluruh data yang belum diverifikasi sengaja ditulis sebagai placeholder. Jangan mempublikasikan angka, nama, status akreditasi, atau dokumen sebelum dikonfirmasi oleh pihak Program Studi Arsitektur.

## Struktur proyek

- `index.html`: homepage prototype
- `assets/css/style.css`: tampilan responsif desktop dan mobile
- `assets/js/script.js`: navigasi mobile dan tahun footer otomatis
- `assets/img/hero-arsitektur.svg`: ilustrasi sementara
- `DATA-YANG-HARUS-DILENGKAPI.md`: checklist bahan resmi yang perlu diminta dari prodi

## Menjalankan secara lokal

Buka `index.html` pada browser atau jalankan menggunakan ekstensi Live Server di Visual Studio Code.

## Menampilkan website melalui GitHub Pages

1. Buka **Settings** pada repository.
2. Pilih menu **Pages**.
3. Pada bagian **Build and deployment**, pilih **Deploy from a branch**.
4. Pilih branch `main` dan folder `/ (root)`.
5. Klik **Save** dan tunggu proses deployment selesai.

## Rencana tahap berikutnya

Setelah homepage disetujui, tambahkan halaman internal untuk profil, visi dan misi, dosen, kurikulum dan CPL, fasilitas, MBKM, berita, pengumuman, serta unduh berkas. Untuk penggunaan resmi jangka panjang, website dapat dipindahkan ke CMS agar admin prodi mudah memperbarui konten.
